## 核心流程

### 引导 & 登录
- [ ] 清除数据后首次启动，显示引导页
- [ ] 完成引导 4 个场景，进入首页
- [ ] 首次启动自动创建 users 文档 (云开发控制台验证)
- [ ] 默认昵称为空（前端显示为"用户"），默认设置 theme=neo, notifyEnabled=true

### 习惯管理
- [ ] 点击添加 → 填写名称 → 保存 → 首页显示
- [ ] 输入违规习惯名称 → 提示"包含违规内容" (内容安全检查)
- [ ] 编辑习惯名称/描述 → 保存成功
- [ ] 归档习惯 → 从活跃列表消失
- [ ] 查看归档列表 → 恢复归档习惯
- [ ] 拖拽排序 → 刷新后顺序保持

### 打卡
- [ ] 点击习惯完成打卡 → 连续天数 +1
- [ ] 再次点击取消打卡 → 连续天数回退
- [ ] 日历上显示打卡圆点
- [ ] 快速连续点击 → 不重复打卡 (防抖)

### 冻结日
- [ ] 使用冻结 → 显示剩余次数
- [ ] 冻结后连续天数不中断
- [ ] 超过月度限额 (2次) → 提示失败
- [ ] 无法冻结非当天日期

### 时间轴
- [ ] 切换到时间轴 Tab → 显示日历
- [ ] 切换月份 → 查看历史打卡记录
- [ ] 过去有打卡的日期显示标记
- [ ] 未来日期显示"还没到来"

### 灵感板
- [ ] 创建文本笔记 → 保存成功
- [ ] 创建清单笔记 → 添加条目 → 保存
- [ ] 输入违规内容 → 提示拦截
- [ ] 编辑笔记 → 修改内容 → 保存
- [ ] 删除笔记 → 确认后消失
- [ ] 下拉刷新 → 数据更新

### 仪式
- [ ] 创建仪式 → 命名 → 关联习惯 → 保存
- [ ] 输入违规仪式名称 → 提示拦截
- [ ] 执行仪式 → 计时器工作
- [ ] 仪式完成 → 关联习惯批量打卡
- [ ] 编辑仪式 → 修改名称/习惯 → 保存
- [ ] 删除仪式 → 确认后消失

### 旅程
- [ ] 浏览旅程列表 → 显示可用旅程
- [ ] 开始旅程 → 进入详情
- [ ] 完成步骤 → 进度更新
- [ ] 完成全部步骤 → 显示完成庆祝页

### 个人资料
- [ ] 查看个人页 → 显示头像/昵称/统计
- [ ] 未完成微信头像同步时进入个人页 → 自动拉起同步面板
- [ ] 未完成微信头像同步时点击个人页二级入口 → 被拦回同步面板
- [ ] 手动填写昵称（仅昵称）→ 保存成功，但仍需继续同步微信头像
- [ ] 微信头像同步（仅头像）→ 保存成功
- [ ] 微信头像同步 + 手动填写昵称 → 保存成功
- [ ] 微信昵称违规 → 提示拦截且资料不变
- [ ] 微信头像上传失败/超时 → 提示失败且可重试
