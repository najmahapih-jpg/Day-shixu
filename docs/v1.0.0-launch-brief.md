# Day时序 v1.0.0 上线简报

## 文档目的

为发布/提审负责人提供上线决策所需的检查清单和入口汇总。本文件不重复发布操作步骤，具体执行流程见 [发布执行指南](RELEASE_GUIDE.md)。

## 版本基线

- **版本号**: 1.0.0（`manifest.json` → `versionName`）
- **AppID**: 见 `project.config.json`
- **云环境**: 见 `cloudbaserc.json`
- **云函数**: ai、board、habit、journey、notify、ritual、stats、user（共 8 个）

## 交付范围概述

| 模块 | 说明 |
|------|------|
| 习惯管理 | 创建、编辑、归档、拖拽排序、内容安全检查 |
| 每日打卡 | 打卡/取消、连续天数、日历标记、防抖 |
| 冻结日 | 冻结保护连续天数，月度限额 2 次 |
| 时间轴 | 月历视图，历史打卡查看 |
| 灵感板 | 文本/清单笔记创建、编辑、删除，内容安全 |
| 仪式 | 创建仪式→关联习惯→计时执行→批量打卡 |
| 旅程 | 预设旅程列表→步骤推进→完成庆祝 |
| 个人资料 | 微信头像/昵称同步、统计展示 |
| AI 洞察 | 周对比分析卡片、建议和口号 |
| 统计详情 | 热力图、连续天数、周对比 |
| 设置 | 减少动效、周起始、习惯提醒、导出数据、清除缓存、意见反馈 |
| 引导流 | 首次启动 4 场景引导 |
| 订阅消息 | 活动进度提醒 + 每日打卡提醒 |
| 版本更新 | getUpdateManager 自动更新提示 |

## 提审前 Go/No-Go 检查

以下全部通过方可提审：

## 微信后台手动配置项

以下必须在 mp.weixin.qq.com 手动完成，无法通过脚本自动化：

1. **隐私协议配置** — 设置 → 服务内容声明 → 用户隐私保护指引
2. **订阅消息模板** — 功能 → 订阅消息 → 确认模板 ID 与代码一致
3. **服务类目** — 设置 → 基本设置 → 服务类目
4. **数据库索引** — 云开发 → 数据库 → 各集合 → 索引管理
5. **数据库自动备份** — 云开发 → 数据库 → 备份与回档 → 每日备份
6. **数据库安全规则** — 各集合设为「仅创建者可读写」

具体配置细节见 [发布执行指南 → 第一步](RELEASE_GUIDE.md)。

## 回滚入口

| 场景 | 操作 |
|------|------|
| 小程序前端回滚 | mp.weixin.qq.com → 版本管理 → 版本回退 |
| 紧急修复 | 上传修复版本 → 提交审核 → 加急审核（每年 1 次） |
| 单个云函数回滚 | 回退 git 到目标版本后 `npm run cf:deploy:one -- <函数名>`（部署的是本地代码，非云端历史版本） |
| 全部云函数回滚 | 回退 git 到目标版本后 `npm run cf:deploy:all` |
